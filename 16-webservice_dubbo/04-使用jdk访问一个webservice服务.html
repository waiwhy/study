2、生成客户端代码
1）wsimport是jdk自带的，可以根据wsdl文档生成客户端调用代码的工具。
2） 无论服务器端的WebService是用什么语言写的，都将在客户端生成Java代码。
3）wsimport.exe位于JAVA_HOME\bin目录下.

常用参数为
  • -d<目录> - 将生成.class文件。默认参数。
  • -s<目录> - 将生成.java文件。
  • -p<生成的新包名> -将生成的类，放于指定的包下。
  • (wsdlurl) - http://server:port/service?wsdl，必须的参数。

示例：命令行输入
d:
wsimport –s . -encoding utf-8 http://127.0.0.1:8089/hello?wsdl

4）注意：
  • 要先发布服务，服务运行的状态下生成代码
  • -s后面有个小点，用于指定源代码生成的目录。点即当前目录。
  • 如果使用了-s参数则会在目录下生成两份代码，一份为.class代码。一份为.java代码。

调用webservice步骤
1）打开WSDL文档
2）从下往上读WSDL文档，先找到service（服务访问点集合），根据Services里面binding属性找到binding元素，再根据binding元素的type属性找到绑定的portType（服务类）
3）根据WSDL的地址生成客户端代码 wsimport -s . http://127.0.0.1:8089/hello?wsdl
4）把客户端代码拷贝到项目中
5）创建服务访问点集合对象
6）根据服务访问点获得服务类
7）调用服务类的方法

5、编写客户端程序
package com.qfedu.client;
import com.yaohuan.ws.MyHello;
import com.yaohuan.ws.MyHelloServer;
public class HelloClient {
    public static void main(String[] args) {
        
        //服务访问点
        MyHelloServer myHelloServer = new MyHelloServer();
        
        MyHello myHello = myHelloServer.getMyHelloPort();
        
        System.out.println(myHello.sayHello("Helen", 18));
        System.out.println(myHello.sayBye("Helen"));
    }
}
