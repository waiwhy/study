2、创建一个服务
package com.qfedu.ws_server_gp;
import javax.jws.WebService;
@WebService
public class HelloService {
    public String sayHello(String name) {
        return "hello " + name;
    }
    public String sayBye(String name) {
        return "bye " + name;
    }
    
}


3、发布服务
package com.qfedu.ws_server_gp;
import javax.xml.ws.Endpoint;
public class HelloServicePublish {
    public static void main(String[] args) {
        
        //jdk发布一个webservice服务
        Endpoint.publish("http://127.0.0.1:8089/hello", new HelloService());
    }
}


4、运行服务程序并访问服务地址
http://127.0.0.1:8089/hello?wsdl

在服务地址中可以发现一些元素：这些元素组成的xml文件叫做WSDL文件


WebService Description Language – Web服务描述语言。
通过XML形式说明服务在什么地方－地址。
通过XML形式说明服务提供什么样的方法 – 如何调用。

WSDL 是基于 XML 的用于描述 Web Services 以及如何访问 Web Services 的语言。


WSDL 文档结构

WSDL 文档是利用这些主要的元素来描述某个 web service 的：
元素
定义
<portType>
web service 执行的操作
<message>
web service 使用的消息
<types>
web service 使用的数据类型
<binding>
web service 使用的通信协议
一个 WSDL 文档的主要结构是类似这样的：

<definitions>

<types> definition of types........
</types>

<message>
definition of a message....
</message>

<portType> definition of a port.......
</portType>

<binding>
definition of a binding....
</binding>

</definitions>

<portType> 元素是最重要的 WSDL 元素。
它可描述一个 web service、可被执行的操作，以及相关的消息。

可以把 <portType> 元素比作传统编程语言中的一个函数库（或一个模块、或一个类）。
<types> 元素定义 web service 使用的数据类型。



5、可以为服务添加一些注解
package com.qfedu.ws_server_gp;
import javax.jws.WebMethod;
import javax.jws.WebParam;
import javax.jws.WebResult;
import javax.jws.WebService;
@WebService(
        serviceName="MyHelloServer",//服务访问点
        portName="MyHelloPort",//端口名
        name="MyHello",//类名
        targetNamespace="ws.yaohuan.com"//名称空间
        )
public class HelloService {
    public String sayHello(String name, int age) {
        return "hello " + name + "。" + age;
    }
    public @WebResult(name="byeResult") String sayBye(//自定义返回结果的名字
            @WebParam(name="name") String name) {//自定义参数名
        return "bye " + name;
    }
    
    
    @WebMethod(exclude=true)//排除发布
    public String sayHi() {
        return "hi";
    }
}
