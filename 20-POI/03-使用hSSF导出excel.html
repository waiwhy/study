测试：WriteExcelHSSF.java
HSSF只操作.xls文件（97-03版本excel），一个sheet中行有限制，最大65536行。
不适合大数据的导出

缺点：
如果数据大，引起内存溢问题。

优点：
编程方便，如果数据量小，写入速度很快。

package name.yaohuan.poi;
/**
 * poi测试导出excel文件，数据量大出现内存溢出
 *
 */
public class WriteExcelHSSF {
    @Test
    public void testWrite() throws IOException {
        
        // 创建文件输出流
        FileOutputStream out = new FileOutputStream("d:/poi-excel/test-write.xls");
        // 创建一个工作簿
        Workbook wb = new HSSFWorkbook();
        Sheet s = wb.createSheet();//创建1个sheet
        wb.setSheetName(0, "人员名单");//指定sheet的名称，第一个参数是sheet的索引
        
        //Sheet s1 = wb.createSheet();//创建第2个sheet
        //wb.setSheetName(1, "人员名单1");
        //xls文件最大支持65536行(测试少量数据和大数据)
        for (int rownum = 0; rownum < 65536; rownum++) {//创建行,.xls一个sheet中的行数最大65535
            // 创建一行
            Row r = s.createRow(rownum);
            for (int cellnum = 0; cellnum < 10; cellnum ++) {//一行创建10个单元格
                // 在行里边创建单元格
                Cell c = r.createCell(cellnum);
                //向单元格写入数据
                c.setCellValue(cellnum + "-" + rownum);
            }
        }
        wb.write(out);//输出文件内容
        out.close();
    }
}
