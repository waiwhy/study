spring boot项目只有src目录，没有webapp目录，会将静态访问(html/图片等)映射到其自动配置的静态目录

static目录是spring boot 默认的静态资源目录


1、创建index.html

可以在resources建立一个static目录和index.htm静态文件，访问地址 http://localhost:8080/index.html


<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
</head>
<body>
<h1>Welcome</h1>
<a href="employee-list.html">员工管理</a>
</body>
</html>

2、加入其他静态资源
 
3、创建employee-list.html
在static文件夹下创建 employee-list.html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Insert title here</title>
    <!-- bootstrap -->
    <link href="public/libs/bootstrap/css/bootstrap.min.css" rel="stylesheet">
    <!-- bootstrap table -->
    <link href="public/libs/bootstrap-table/bootstrap-table.min.css" rel="stylesheet" >
</head>
<body>
    <table id="table"></table>
    <!-- 全局js -->
    <script src="public/libs/jquery.min.js"></script>
    <script src="public/libs/bootstrap/js/bootstrap.min.js"></script>
    <!-- bootstrap table -->
    <script src="public/libs/bootstrap-table/bootstrap-table.min.js"></script>
    <script src="public/libs/bootstrap-table/locale/bootstrap-table-zh-CN.min.js"></script>
    
    <script>
    
    $('#table').bootstrapTable({
        url: 'employee/list',
        pagination: true,    //显示分页条  
        showRefresh: true,  //显示刷新按钮
        search: true,
        striped : true,     //设置为true会有隔行变色效果 
        columns: [{ title: '员工ID', field: 'id'},
            { title: '姓名', field: 'name'},
            { title: '地址', field: 'address'},
            { title: '照片', field: 'photo'}
        ]});
    </script>
</body>
</html>
