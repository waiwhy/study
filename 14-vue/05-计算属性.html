1、计算属性：computed  demo01

<div id="app">
    <p><input type="text" v-model="message" /></p>
    <p>{{ message }}</p>
    <p>{{ reversedMessage }}</p>
</div>
<script src="../script/vue.min.js"></script>
<script>
new Vue({
    el: '#app',
    data: {
        message: 'hello vue',
        ok: false
    },
    computed: {
        reversedMessage: function(){
            return this.message.split('').reverse().join('');
        }
    }
});
</script>


2、计算属性 VS 方法

computed ：依赖于缓存，只有相关依赖发生改变时，才重新取值（性能好，但是占用缓存）
method：在每次重新渲染页面时，都会被执行（如果不希望使用缓存，可以选择method）
<div id="app">
    <p><input type="text" v-model="message" /></p>
    <p>{{ message }}</p>
    <p>{{ reversedMessage }}</p>
    <p>{{ reversedMessage2() }}</p>
</div>
<script src="../script/vue.min.js"></script>
<script>
new Vue({
    el: '#app',
    data: {
        message: 'hello vue',
        ok: false
    },
    methods: {
        reversedMessage2: function(){
            return this.message.split('').reverse().join('');
        }
    },
    computed: {
        reversedMessage: function(){
            return this.message.split('').reverse().join('');
        }
    }
});
</script>
