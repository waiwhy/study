1、定义自定义异常
exception包下创建LackParamException.java

2、调用自定义异常
        if(StringUtils.isEmpty(uidStr)) {
            //拋异常
            throw new LackParamException();
        }
        if(StringUtils.isEmpty(lockStr)) {
            //拋异常
            throw new LackParamException();
        }

3、在web.xml中配置异常处理节点：error-page
可以针对异常的类名配置，也可以针对http错误码配置

4、创建异常页面
如500.jsp、404.jsp
注意：如果要在页面中显示错误信息，则需开启jsp页面的 isErrorPage属性

5、系统中的其他异常处理
能抛出的一律抛出，如果因为实现接口等原因无法抛出，则catch后 重新抛出自定义异常，因为使用了servlet，
所以自定义异常对象如果想要成功抛出，必须继承ServletException
事务处理的代理中需要捕获异常完成正常的事务处理逻辑，然后重新抛出新的自定义异常对象
