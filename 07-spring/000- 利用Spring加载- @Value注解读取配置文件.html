二：在spring配置文件中配置加载上述的资源文件。（两种方式）

方式1：s

[java] view plain copy
  1. <context:property-placeholder location="classpath:config/*.properties"/>  

方式2；

[java] view plain copy
  1. <bean id="propertyBean"  
  2.     class="org.springframework.beans.factory.config.PropertyPlaceholderConfigurer">  
  3.     <property name="locations">  
  4.         <array>  
  5.             <value>classpath:config/*.properties</value>  
  6.         </array>  
  7.     </property>  
  8. </bean>  

三：使用配置文件中配置的值

[java] view plain copy
  1. @Value("${address}")    
  2. private String ftpAddress;   

这时ftp的地址直接就注入到ftpAddress中。
